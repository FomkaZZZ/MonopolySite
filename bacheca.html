Правила игры

здесь вы можете ознакомится с основными механиками монополии в майнкрафт

Основы

1.Начало игры.

Игроки ходят по очереди, бросая 2 кубика и передвигаясь на выпавшее количество клеток.
При дубле (одинаковые числа) игрок делает ещё один ход, но три дубля подряд отправляют его в тюрьму.

2.Покупка недвижимости.

Если игрок попадает на свободную улицу, он может её купить у банка по номиналу. Если игрок отказывается покупать, недвижимость остается в продаже.

3.Арендная плата.

Если игрок попадает на чужую собственность, он платит аренду.

4.Строительство.

Чтобы строить дома, нужно владеть всеми улицами одного цвета. Максимум 4 дома, затем можно построить отель. Улицы с поставленными домами/отелями передовать нельзя.

5.Торговля.

Игроки могут между собой обмениваться купленым имуществом.

6.Тюрьма.

Выйти из тюрьмы можно:
Заплатив 50M.
Использовав карту "Освобождение".
Попытавшись выбросить дубль (3 попытки, затем плата за выход).

7.Банкротство.

Если игрок не может заплатить, он объявляет банкротство и выбывает. Все имущество поступает в продажу.

8.Конец Игры.

Игра продолжается, пока все, кроме одного, не обанкротятся.

Механики игры

Чтобы бросить кубик нажмите "Q".

Для следующих взаимодействий использовать "ПКМ":
Шлем- для трейда с игроками.
Бумага - управление компаниями.
Компас - телепорт на место последней остановки.
Кровать - сдаться/выйти.
Книга - Для взаимодействий с игрой(покупка улиц, плата за аренду, оплата услуг)
Наковальня - Для выхода из тюрьмы (заплатить 50М, с помощью дубля, использовать освобождение)
Карта - показывает мини карту поля, можно посмотреть какие поля заняты или нет.
